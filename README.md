## Cara Penggunaan
Dibuat menggunakan framework Next.js dengan bantuan NextUI sebagai library.

First, run the development server:
```bash
npm  run  dev
# or
yarn  dev
# or
pnpm  dev
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
